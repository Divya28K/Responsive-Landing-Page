## Responsive Landing Page

## Overview  

FlowTask is a responsive marketing landing page built using Next.js + Tailwind CSS, designed to showcase an AI-powered task management platform for teams.

# Live Demo

Check out the live site here:

https://v0-responsive-landing-page-lyart-delta.vercel.app/

## Structure  

- Next.js (App Router) for pages & routing

- Tailwind CSS for styling and responsive layout
  
- Organized components: Hero, Features, Testimonials, Pricing, Footer

## How to run locally  

1. Clone the repo https://github.com/Divya28K/Responsive-Landing-Page.git
   
2. npm install
   
3. npm run dev — open http://localhost:3000
   
4. After changes, push to main → Vercel auto-deploys
